我要先打一份內文 如果要有粗體 就是這樣 增加2個星號 就是

斜體 如果要斜體 就是這樣變兩個星號而已

有序序列

  1. 就是前面這樣 如果下面還要加的話
  2. 第一種加法是你直接打2.這樣即可
  3. 或是直接打一個 1. markdown 會自動去計算第幾個

無序序列

  • 直接打一個星號 空白鍵
  • 這個主題比較屌 旁邊會有小手

無序序列 與 有序序列是可以階層的

  • 假如要內縮一個無序序列的話
    • 先打4個 空白鍵 在家 *
  1. 有序序列也是
    1. 先打4個空白鍵 在打1. 即可
    2. 先打4個空白鍵 在打1. 即可
    3. 如果又要內縮一層的話
      1. 再打4個空白鍵 打1. 即可
        1. 以此類推

接下來是引用

先打>的符號 再打空白鍵

超連結

想要呈現的文字用中括號保起來小括號放網址

如果想要超連結是個人網站其他文章的話

假設我想要呈現post1的文章 一樣小括號(/blog/post-1)

假設我想要呈現post1的文章

圖片 這是外部的連結

跟剛剛的超連結有點像

如果想要用資料夾的圖片的話 不是外部連結

你需要這樣打 跟呈現文章有一點相似 ! [] () 記得副檔名有要打上

假如想要相簿裡面的資料夾 就是階層的概念而已

單行程式碼

如果要打上單行程式碼的話只要在程式碼兩邊打上兩個小引號 print("Hello Wolrd")

多行程式碼 開頭輸入3個小的引號 結尾也是

public class HomeWork {
	public static void main(String args[]){
		
	}
}

更屌的來了 在多行程式碼 後面可以打上程式語言名稱 例如 java / python

直接在開頭的3個小引號打程式語言的名稱即可

public class HomeWork {
	public static void main(String args[]){
		System.out.println("Java");
	}
}
